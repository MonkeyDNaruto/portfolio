Latest Project

Word Master

Word Master is an interactive word-guessing game where players attempt to guess a hidden word within a limited number of attempts. I developed key features such as the dynamic word validation system, responsive UI, and game logic that provides feedback on correct and incorrect guesses. The game includes a navigation bar that stays perfectly aligned on laptop screens and a pop-up screen to display win/loss messages along with a restart button. Try it out here!
